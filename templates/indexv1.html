Make News Credible Again #MNCA

Leveraging Machine Learning to Detect Fake News

W210 - Capstone
Sashi, Brennan, Talieh, and Umber

Project completed as a group using open and preexisting secondary data with the goal of combining technical, analytic, interpretive, and social dimensions to design and execute a full data science project with a focus on real-world applications and situations. Our Goals: Build a model to discern the credibility of an article based on its textual content. Impact the credibility of public discourse and alleviate the strained resources of fact-checkers at a pace scalable with the size of the global internet. Prepare for careers in data science while also doing our part to societally improve the world we live in.

Our Approach to Making News Credible Again

High-level Overview:
1.) Collect news feeds from a set of different sources for either credible or non-credible sites

2.) Custom web scrapers running on the server background every night that fetches each article as a text file in html format

3.) Once the data is collected and stored, ingest it in a pandas dataframe. We first preprocess the texts it using text preprocessing tokenization and then apply different algorithms, including Multinomial Naive Bayes, Support Vector Machine, Logistic Regression & XGBoost classifier to optimize prediction capability

4.) To also extract interactive insights, preprocess using the tfidf algorithm and then cluster it using 2 different algorithms: K-means and Latent Dirichlet Allocation (LDA), after which we visualize the resulting clusters using two interactive python visualization libraries: Bokeh and pyldavis
